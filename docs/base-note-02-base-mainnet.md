# Base Mainnet Overview

Base Mainnet is the live Base network where real assets are used.

Useful points:
- Gas is paid with ETH.
- Contract deployments cost real ETH.
- Transactions are permanent after confirmation.
- Always check that MetaMask is connected to Base.
- Save transaction hashes after important actions.

Base is useful for low-cost EVM activity, tokens, NFTs, and smart contract testing.
